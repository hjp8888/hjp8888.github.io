📥 티들리위키에 Import하는 방법
1. 위에서 JSON 다운로드 버튼 클릭
2. 티들리위키(index.html 등) 브라우저에서 열기
3. 우측 상단 ≡ 메뉴 → Import 클릭
4. 다운받은 .json 파일 선택
5. 티들러 목록 확인 후 Import all 클릭
* frontmatter의 title, tags가 자동으로 적용됩니다.
* title이 없으면 파일명이 title로 사용됩니다.
